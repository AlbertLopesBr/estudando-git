git bash here = dentro da pasta pra iniciar
git init = dentro do prompt para iniciar
git status = para ver o status dos arquivos dentro da pasta
git add "o nome do arquivo.extensao" =  para listar o arquivo da pasta
git add . = para listar todos os arquivos da pasta
git commit -m "commit inicial" = para iniciar primeira versão do arquivo
git push --set-upstream origin master = enviar os comandos pra nuvem
git remote add origin e escreve url do repositório = define para onde o codigo será enviado na núvem

no commits yet significa que não há versões do código/arquivo
no primeiro acesso lista o email e nome do usuário da conta git


fiz a modificação no código

dei um git status para verificar que aparece a mensagem em vermelho modified: os arquivos
então dou um git add . para adicionar esses arquivos ao prompt
em seguida dou um git commit -m "informo a mudança feita"
por ultimo dou um git push
dou f5 no site, clico na minha branch em main e master e verifico se subiu com a mudança feita
